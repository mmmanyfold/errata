# Errata

### if you are here you know what's up

### Development:

- install boot from http://boot-clj.com/
- run $ boot dev
- visit localhost:3000

### Browser Connected REPL:

- boot dev
- boot repl --client
- (start-repl)

### Building Static Assets for Deployment:

- boot production build target

### Testing:

- boot test

### Running test on save: 

- boot auto-test
